# Security Policy

## Supported versions

Only the latest `0.x` release receives security fixes.

| Version | Supported |
|---------|-----------|
| 0.x (latest) | Yes |
| older | No |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Use GitHub's built-in private reporting:
[Report a vulnerability](https://github.com/Glitni/dlt-saga/security/advisories/new).
This creates a private thread visible only to the maintainers — no email required.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions (if known)

## Response timeline

This is a small open source project maintained on a best-effort basis. We will
aim to acknowledge reports within a week and coordinate a fix before any public
disclosure. Reporters are credited in the release notes unless they prefer to
remain anonymous.
